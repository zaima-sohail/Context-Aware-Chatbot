Artificial Intelligence (AI) is the simulation of human intelligence by machines.

Machine Learning is a subset of Artificial Intelligence.

Deep Learning is a subset of Machine Learning that uses neural networks.

Retrieval-Augmented Generation (RAG) combines document retrieval with Large Language Models.

FAISS is a library developed by Meta for similarity search and vector retrieval.

Streamlit is a Python framework used to build web applications for Machine Learning and AI.
